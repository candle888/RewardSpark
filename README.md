# RewardSpark Prototype

A simple mobile-first web prototype for a rewards app concept where users earn money through:

- Installing partner apps
- Watching short advertisements
- Completing lightweight offers

## Included features

- Interactive task completion with simulated wallet growth
- Security center with easy-to-manage protection toggles
- Detailed combined Terms of Service and Privacy Policy page

## Run locally

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
